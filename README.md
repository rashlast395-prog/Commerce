# Yussif Eats – Fast Food & Restaurant E-Commerce
#### Preview

 - [Live Demo](https://abdulrashidyussif.github.io/yussif-eats/)

#### About

This is a fully functional fast food e-commerce website built with HTML, CSS, and JavaScript. It features a complete ordering system with cart, multiple payment methods, and order tracking.

## Getting Started

1. Clone Repository
```
git clone https://github.com/abdulrashidyussif/yussif-eats.git
```

2. Open `sarab/index.html` in your browser

## Features

- Full e-commerce shopping cart with quantity controls
- Multiple payment methods (Mobile Money, Bank Transfer, Card, Cash on Delivery)
- Order confirmation with live tracking
- Menu filtering and search
- Product detail popups with nutrition info
- Table reservation system
- Contact form
- Newsletter subscription
- Responsive design for all devices
